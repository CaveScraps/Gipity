# Gipity

A CLI application for sending queries to ChatGPT.


## Installation

Clone the repo, install dependencies and build.

```bash
  git clone https://github.com/Christopher-Ayre/Gipity.git
  cd Gipity
  go mod tidy
  go build -o gpt
```

## Usage:

```bash
  ./gpt "Your question goes here"
```
